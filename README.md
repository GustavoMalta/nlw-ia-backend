## **Run Backend project**

\
Download dependences:

```
npm install
```

or

```
yarn
```

---

\
Prepare Database:

```
npx prisma migrate dev
```

or

```
yarn prisma migrate dev
```

---

\
Execute Project:

```
npm run dev
```

or

```
yarn dev
```

---

\
Check Database with Prisma:

```
npx prisma studio
```

or

```
yarn prisma studio
```
